# 🛡️ Sistema de Autenticação Dual

Este projeto apresenta uma interface moderna e responsiva para sistemas de login e cadastro, focada em uma experiência de usuário (UX) fluida e visualmente atraente.

> 🚧 **Status do Projeto:** Em andamento (Building...) 🚧

## 📋 Sobre o Projeto
A interface utiliza um layout de painel duplo que permite ao usuário visualizar as opções de acesso e criação de conta simultaneamente. O diferencial é o uso de interatividade via JavaScript para destacar a seção ativa.

## 🚀 Tecnologias Utilizadas
*   **HTML5:** Estrutura semântica dos formulários.
*   **CSS3:** Design responsivo, uso de Flexbox e animações de transição.
*   **JavaScript:** Manipulação dinâmica de classes para efeitos de foco e desfoque.

## 🛠️ Funcionalidades e Progresso
- [x] Interface lado a lado (Login/Cadastro).
- [x] Efeito de destaque visual interativo.
- [x] Design responsivo para diferentes telas.
- [ ] Validação de formulários em tempo real.
- [ ] Conexão com banco de dados.

## 🔧 Como utilizar
1. Baixe ou clone os arquivos do repositório.
2. Certifique-se de que o arquivo `style.css` e o script `js` estão nas pastas corretas.
3. Abra o `index.html` em qualquer navegador moderno.
